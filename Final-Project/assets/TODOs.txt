Python function flow:
    py_sym(data) -> C_sym(data) -> calculates & returns A
    py_ddg(data) -> C_ddg(data) -> calls C_sym(data) -> calculates & returns D
    py_norm(data) -> C_norm(data) -> calls C_ddg(data) -> calculates & returns W
    py_symnmf(H_0, W, N, d, K) -> C_symnmf(H_0, W, N, d, K) -> calculates & returns H

*** N, d - are inferred in C_sym()
*** K is inferred in Python (args) and passed along to py_symnmf()
*** calling one of sym/ddg/norm in C automatically creates the previous prerequisite matrices and saves them in global pointers.
